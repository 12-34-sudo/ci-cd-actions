# CI/CD GitHub Actions Project

This project demonstrates a **CI/CD pipeline using GitHub Actions**.

## Project Overview
- Simple Python app that prints a message.
- Fully automated CI/CD workflow with GitHub Actions.
- Shows build, install, and run steps in a pipeline.

## How CI/CD Works
1. **Push code** to GitHub.
2. **GitHub Actions** triggers automatically.
3. Python environment is set up.
4. Dependencies installed.
5. App is executed automatically.
6. Build output is visible in Actions logs.

## Tech Stack
- Python 3.11
- GitHub Actions
- Automation / CI/CD

## Why this project matters
- Demonstrates ability to automate builds and runs.
- Recruiters can see your CI/CD knowledge in action.
- Fully web-based: No Python or Docker needed locally.
